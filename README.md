# Команды для управления файлами из терминала

**cd *name***- перемещение в папку

**cd ..**- перемещение на уровень назад

**new-item *name.zyx*** - создать файл с разрешением

**mkdir *name*** - создать папку

**ls** - просмотр файлов в текущей папке 

# Работа с ветками

**git branch** - вывод списка веток

**git checkout *branch_name*** - переместиться в ветку

**git branch *branch_name*** - создание новой ветки

**git -d *branch_name*** - создание и переход в новую ветку

**git merge *branch_name*** - смерджить ветку в текущую

**git branch -d *branch_name*** - удалить слитую ветку

**git branch -D *branch_name*** - удалить НЕ слитую ветку 
# Работа с удаленными репозиториями

**git remote** - выводит список доступных удаленных репозиториев

**git clone *repo_url*** - клонирует указанный удалённый репозиторий

**git remote add *shortname* *url*** - добавить удаленный репозиторий и присвоить ему имя

**git fetch *repo_url*** - загрузить удаленный репозиторий в локальный(без мерджа)

**git pull *repo_url*** - fetch+merge c текущим состоянием


**git commit -m *comment*** - сохранение в локальный репозиторий с комментарием

**git push** - отправка комита в удаленный репозиторий
